# Teste de Hipóteses com Data Science

Projeto desenvolvido durante o curso **"Data Science: Testando Hipóteses"**, com foco em análise estatística aplicada para tomada de decisão baseada em dados.

## Objetivo

Aplicar testes estatísticos para validar hipóteses em cenários reais, utilizando dados para responder perguntas de negócio.

## Conceitos Aplicados

- Teste de hipóteses
- Hipótese nula (H0) e alternativa (H1)
- Testes estatísticos (t-test, Mann-Whitney, qui-quadrado, etc.)
- Interpretação de p-valor
- Análise comparativa entre grupos
- Tomada de decisão baseada em evidências

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- SciPy
- Matplotlib / Seaborn
- Jupyter Notebook / Google Colab

## Estrutura do Projeto

```
data-science-testes-hipoteses/
│
├── notebook.ipynb
├── README.md
└── requirements.txt
```

## Análise Realizada

Durante o projeto, foram analisados cenários como:

- Comparação entre grupos independentes
- Avaliação de impacto antes e depois (amostras pareadas)
- Validação estatística de mudanças em processos

Cada análise segue o fluxo:

1. Definição do problema
2. Formulação das hipóteses
3. Escolha do teste estatístico
4. Execução do teste
5. Interpretação dos resultados

## Principais Insights

- Nem toda diferença observada é estatisticamente significativa
- A escolha correta do teste é essencial para conclusões válidas
- Dados precisam ser analisados com rigor antes de decisões
