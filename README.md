# Teste de Hipóteses com Data Science

Projeto desenvolvido durante o curso **"Data Science: Testando Hipóteses"**, com foco em análise estatística aplicada para tomada de decisão baseada em dados.

## Objetivo

O objetivo deste projeto é aplicar testes estatísticos para validar hipóteses em cenários reais, utilizando dados para responder perguntas de negócio.

## Conceitos Aplicados

- Teste de hipóteses
- Hipótese nula (H0) e alternativa (H1)
- Testes estatísticos (t-test, Mann-Whitney, qui-quadrado, etc.)
- Interpretação de p-valor
- Análise comparativa entre grupos
- Tomada de decisão baseada em evidências

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- SciPy
- Matplotlib / Seaborn
- Jupyter Notebook / Google Colab
